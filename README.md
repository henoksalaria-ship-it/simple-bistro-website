# Simple Bistro Premium Website

A Vite + React + Tailwind + Framer Motion website for Simple Bistro.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL Vite gives you, usually http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Assets

Client images are in:

```txt
public/images/
```

Current local assets:
- simple-bistro-logo.png
- texas-style-burger.png
- chicken-beef-junkie.png
- chicken-junkie.png
- saucy-burger.png

## Deploy

Push this folder to GitHub and import the repo into Vercel.

Vercel settings:
- Framework: Vite
- Build command: npm run build
- Output directory: dist
